# preprocesamiento-ciencia-datos
## Objetivo
Aplicar de manera efectiva el uso de Git y GitHub para la gestion de versiones y la colaboracion en proyectos de Ciencias de Datos
---
##Estructura basica del proyecto

preprocesamiento-ciencia-datos/
|
|--data/# Archivos de datos originales
|--notebooks/# Cuadernos Jupyter o Google Colab (.ipynb)
|--scripts/# Scripts Python reutilizables (.py)
|--outputs/# Resultados o datasets procesados
|--gitignorre # Archivos y carpetas que no se subirán al repositorio
|__README.md #Descripcion general del proyecto
