# PROCESAMIENTO DE DATOS EN CIENCIA DE DATOS

## INTRODUCCION 

El presente proyecto tiene como objetivo aplicr de manera practica el uso de GIT Y GITHUB para gestionar de mejor forma el procedimiento del proyecto y la colaboaracion en proyectos centrados en ciencias de datos, nos basamos en codigo python 

## COMANDOS GIT USADOS 

- git clone URL | Clona el repositorio remoto de GitHub en el equipo local para poder trabajar con sus archivos 
- git config --global user.name "Tu Nombre" | Configura el nobre de usuario
- git config --global user.email "tu_correo@example.com" | Configura el correo electronico asociado a la cuenta de GitHub
- git add "archivo" | Agrega archivos para incluirlos en el proximo commit
- git add preprocesamiento.py | Agrega el rchivo con el codigo Python para trabajar en el 
- git commit -m "mensaje" | Registra un punto de control. El mensaje debe describir el cambio realizado
- git commit -m "Agregar script de preprocesamiento de datos con Pandas"
- git push origin main | Envia los commits locales a la rama 'main' del repositorio 
- git push origin feature-preprocesamiento
